Key Files to Understand
========================

1. app/page.tsx
   - Main dashboard page
   - Fetches data from API
   - Manages filters and state
   - Renders metrics, table, and panel

2. components/LeadTable.tsx
   - Displays leads in sortable table
   - Handles inline editing
   - 13 columns with various data types

3. components/LeadDetailPanel.tsx
   - Slides in from right
   - Shows complete lead details
   - Editable CRM fields
   - Score breakdown visualization

4. app/api/leads/route.ts
   - GET: Fetch all leads from JSON
   - PATCH: Update specific lead
   - File system operations

5. lib/types.ts
   - TypeScript interfaces for all data
   - Lead, Company, Qualification, etc.
   - Type safety throughout app

6. lib/utils.ts
   - Industry classification
   - Lead ID generation
   - Date formatting
   - Color mapping
   - Clipboard operations


Data Flow
=========

1. Python Pipeline
   └─> data/leads.json (one level up from dashboard)

2. Next.js API Route
   └─> app/api/leads/route.ts
       ├─> GET: Read leads.json
       └─> PATCH: Update leads.json

3. React State
   └─> app/page.tsx
       ├─> useState: leadsData, filters, selectedLead
       └─> useMemo: tableRows, filteredLeads, metrics

4. UI Components
   └─> components/LeadTable.tsx
   └─> components/LeadDetailPanel.tsx
